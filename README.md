# Orlev Press-Ons

The Art of Digital Couture — Premium press-on nails e-commerce platform.

## Tech Stack

- **Frontend:** Next.js (React, Vite)
- **Backend:** Node.js (Express)
- **Database:** Prisma + SQLite
- **Styling:** Tailwind CSS

## Setup

### Backend

```bash
cd backend
npm install
npx prisma generate
npx prisma db push
npm run dev
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```
